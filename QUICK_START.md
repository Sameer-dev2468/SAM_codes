# 🌤️ Weather Prediction App - Quick Start Guide

## 🚀 **Easy Ways to Start the App:**

### **Option 1: Double-Click Method (Easiest)**
1. **Double-click** `start_weather_app.bat` 
2. The app will automatically:
   - Check if Python is installed
   - Install required packages if needed
   - Start the Flask server
3. **Open your browser** and go to: `http://localhost:5000`

### **Option 2: PowerShell Method**
1. **Right-click** `start_weather_app.ps1`
2. Select **"Run with PowerShell"**
3. The app will start with colorful status messages
4. **Open your browser** and go to: `http://localhost:5000`

### **Option 3: Manual Method**
1. Open Command Prompt or PowerShell
2. Navigate to the project folder
3. Run: `python weatherpred.py`
4. **Open your browser** and go to: `http://localhost:5000`

## 📱 **Access the App:**
- **Local:** http://localhost:5000
- **Network:** http://192.168.29.235:5000

## 🛑 **How to Stop the App:**
- Press **Ctrl+C** in the terminal/command prompt
- Or close the terminal window

## 🔧 **Troubleshooting:**

### **If Python is not found:**
- Download and install Python from: https://python.org
- Make sure to check "Add Python to PATH" during installation

### **If packages are missing:**
- The launcher will automatically install them
- Or manually run: `pip install -r requirements.txt`

### **If port 5000 is busy:**
- The app will show an error
- Close other applications using port 5000
- Or modify the port in `weatherpred.py`
